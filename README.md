# K-Fashion-Project

K-Fashion image를 학습한 모델을 구현하고 모델을 통하여 사용자가 원하는 옷의 스타일을 분류해주고
그 스타일과 비슷한 옷을 추천 및 옷의 정보를 제공해주는 서비스
